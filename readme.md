## 📦️ Como rodar o projeto

Clone o projeto com o comando abaixo:

```bash
  # Clone o repositório
  ❯ git clone git@github.com:VanilsonJunior/OOP-Senai.git
# Entre no diretório
  ❯ cd [ProjetoSenaiOOP]
# Execute o projeto
  ❯ dotnet run
```
